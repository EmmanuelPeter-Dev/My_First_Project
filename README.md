# Airbnb Clone

A responsive Airbnb-inspired frontend website built using HTML and CSS. This project was created to demonstrate modern web design, responsive layouts, and user interface development by recreating the look and feel of the Airbnb website.

## Features
- Responsive landing page
- Login page
- Help Center page
- Navigation bar linking all pages
- Clean and modern user interface
- Mobile-friendly layout

## Technologies Used
- HTML5
- CSS3
