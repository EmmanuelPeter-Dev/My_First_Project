# My_First_Project
This project is a responsive frontend clone of the airbnb website, built to strengthen my HTML and CSS skills through real-world practice. The goal was to recreate Airbnb's modern user interface while focusing on clean code, responsive layouts, and attention to details.
THE FEATURES
Responsive landing page
Login page
Help Center page
Modern and clean user interface
Mobile-friendly design
Built with semantic HTML5 and CSS3
PROJECT STATUS
This project currently focuses on the frontend design. JavaScript functionality has not yet been implemented and will be added in future updates to make the website more interactive and closely match the original Airbnb experience.
PURPOSE
This project was created as part of my frontend web development learning journey to improve my ability to build responsive, real-world websites and strengthen my UX/UI implemetation skills.
